## Getting Started

No build tools required. Simply open `index.html` in any modern browser:

```
double-click index.html
```

Or serve locally with any static file server:

```bash
# Using Python
python -m http.server 8080

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8080`.
